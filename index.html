Love Letter Typewriter

A font that had been used in "lain"

This font is based on what Chiaki J. Konaka has published here.

I fixed some warnings in the font and converted it to a format that can be used as a web font.

You can see a web font demo here.

You can download fonts from the GitHub repository.

  • * LoveLetterTypewriter.ttf : A modified font from which web fonts are generated
  • * webfontkit/ : Generated by Webfont Generator
  • * originals/ : Original resources from Chiaki J. Konaka's website.

This font currently does not contain many glyphs.

If you give me glyph vector data, I will add it to the font. Of course, you are welcome to add to the ttf file.
